# Sentiment Analysis Using BiLSTM

## Project Overview

This project performs sentiment analysis on movie reviews using a Bidirectional Long Short-Term Memory (BiLSTM) neural network.

The model classifies reviews as:
- Positive
- Negative

## Dataset

IMDb 50K Movie Reviews Dataset

- 50,000 movie reviews
- 25,000 positive reviews
- 25,000 negative reviews

## Technologies Used

- Python
- Pandas
- TensorFlow
- Keras
- Scikit-Learn
- Google Colab

## Data Preprocessing

- Label Encoding
- HTML Tag Removal
- Lowercase Conversion
- Tokenization
- Sequence Padding

## Model Architecture

- Embedding Layer
- Bidirectional LSTM Layer
- Dense Output Layer

## Results

**Test Accuracy: 88.93%**

## Conclusion

A BiLSTM-based sentiment analysis model was trained on the IMDb 50K Movie Reviews dataset and achieved 88.93% accuracy.
